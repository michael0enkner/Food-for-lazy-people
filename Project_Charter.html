Project Charter

Project title

Food for lazy people
because we want that lazy people or people which have not much time to go in their kitchen and make nevertheless of time
and idleness some easy food. We hope that we can inspire these people for cooking!

Client

Herr Professor Klewein
This is a project which we have to do in school in the subject "Webtechnik".

Goals

Program goals

  • Sign in System
    • This is as the name says a sign in system. You need to sign in to use the rating system and if you want to add recipes.
  • Add recipes
    • You can add recipes there.
  • Ratingsystem
    • This is a system where you can rate the recipes, the best is a one and the worst is a five.
  • Search
    • You can search by the following criteria: vegan, vegetarian, time.
    With vegan and vegetarian, we mean that you can pick a checkmark and then only recipes with these criteria come.
    With time we mean that you can choose how long the recipe should take to cock.
  • Filtering system
    • In this system you can choose some ingrediances and then you get these recipes with the chosen ingrediances.

    Goals for users

  • inspire for cooking
  • try new recipes
  • show your recipes
  • make cooking easier for some people
